# 核心用户流程

- 状态：`draft`

## 学生首次学习

```text
选择学段和兴趣
→ 完成简短前测
→ 进入推荐课程
→ AI教师说明目标
→ Canvas展示内容
→ 学生操作或答题
→ 即时反馈
→ 学习总结与下一步建议
```

## 学生自由提问

```text
提出问题
→ 识别学段与意图
→ 检索审核知识
→ 选择讲解形式
→ 对话或Canvas回答
→ 追问理解情况
→ 记录学习事件
```

## 教师上传教材

```text
上传文件
→ 安全扫描
→ OCR与版面解析
→ 章节和知识点切分
→ Embedding与索引
→ 抽样审核
→ 发布到指定课程
```

## 答错后的补救

```text
识别错误类型
→ 判断是知识缺失还是表达错误
→ 切换讲解方式
→ 给出更小步骤
→ 再次练习
→ 更新掌握度
```
