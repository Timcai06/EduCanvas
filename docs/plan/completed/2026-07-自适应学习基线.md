# 自适应学习基线

- 状态：`completed`
- 负责人：项目负责人
- 最后验证时间：2026-07-24
- 完成时间：2026-07-24

## 目标

交付年级自适应学习的 P0/P1 纵切：先固定学习者画像与诊断证据边界，再让学生从 Web 显式建立 Notebook 学习目标、完成确定性短诊断，并进入使用可信进度和有限教学偏好的统一学习工作区。

## 交付结果

- [x] ADR 固定显式年龄/年级、可修改教学偏好和未成年人默认安全策略；
- [x] 固定 Notebook Goal、6–12 节点目标图、公开诊断与服务端答案分层；
- [x] 固定 Attempt 重放、归属拒绝、可信学习事件与 mastery 写入规则；
- [x] 新增数据/API 契约并回写产品、架构、数据和安全文档；
- [x] `teaching-core` 提供画像、课程图、公开投影和确定性诊断纯逻辑；
- [x] PostgreSQL 保存画像、Goal、Objectives、Attempt 与 Responses；
- [x] 诊断事务原子写 `assessment_graded` 与 mastery，并覆盖重放/冲突/跨学生测试；
- [x] Web 提供画像与目标表单、短诊断、三态目标进度和学习工作区；
- [x] Teaching Profile 消费有限适配策略，不接收人格标签；
- [x] 全量 lint/typecheck/unit/integration/build/E2E 和亮暗主题人工目检。

## 非目标

- 不从文本、图片、声音或行为推断年龄、性格、心理与能力；
- 不实现课程编辑器、模型生成题库或教师发布工作流；
- 不实现 P2 的动态图推荐、目标切换、间隔复习与长期 Memory；
- 不引入 LangGraph、新 Agent Loop、新 Tool Runtime 或 Redis；
- 不改 TUI 与渠道入口。
