# 项目简报

- 状态：`accepted`
- 来源：官方赛题细则与团队讨论

## 赛题

JBGS-2026-02：多模态K12人工智能通识课教学助手对话智能体。

## 产品目标

构建一个面向小学到高中学生的在线AI教师。系统根据学生学段、教材、学习历史和当前表现，选择合适的知识深度、语言和教学形式。

## 必须形成的闭环

```text
选择学段或课程
→ 判断当前水平
→ 对话讲解与主动引导
→ Canvas互动学习
→ 练习或编程实践
→ 自动评价与反馈
→ 更新学生掌握度
→ 推荐下一步内容
```

## 赛题要求映射

- 年级自适应；
- 至少三种多模态交互；
- 个性化学习路径；
- 可运行系统原型；
- API、页面嵌入或其他部署集成方案；
- 完整项目报告。

## 北极星目标

在缺少专业AI教师的情况下，学生仍能独立完成一节准确、有趣、可操作、有反馈的AI通识微课。

## 当前示范主题建议

“AI如何识别猫和狗”：同一知识点分别用绘本、分类游戏、特征可视化和Python实验适配不同学段。
