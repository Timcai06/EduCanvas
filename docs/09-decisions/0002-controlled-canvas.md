# ADR-0002：采用受控教学Canvas

- 状态：`superseded`
- 日期：2026-07-13
- 取代者：[ADR-0010](0010-canvas-trust-tiers.md)——判分型 Artifact 机制原样保留为 Tier 1；新增沙箱探索型 Tier 2，"主页面不执行模型代码"的不变量继续有效

## 背景

产品希望获得类似Gemini Canvas的生成式体验，但面向未成年人，需要稳定、可评测、可访问且安全的教学内容。

## 决定

模型输出版本化的Canvas Artifact Schema，前端将其渲染为审核过的React组件。动画由GSAP组件执行。模型生成的任意HTML、JavaScript或GSAP源码不在主页面直接运行。

## 原因

- 防止代码注入；
- 保证教学组件视觉和交互质量；
- 便于自动测试和回放；
- 支持记录细粒度学习事件；
- 保持不同模型之间的输出稳定性；
- 可以逐步扩展新的Artifact类型。

## 后果

- 需要维护Artifact Schema和组件注册表；
- 新教学形态需要先实现组件；
- 模型自由度降低，但产品可靠性、性能和安全性提高。
