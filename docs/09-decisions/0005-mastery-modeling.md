# ADR-0005：掌握度建模与误区标注

- 状态：`accepted`
- 日期：2026-07-14
- 依据：团队深度研究（KT 文献综述 + K12 ML 误区研究 + 掌握学习证据），详细规格见 `docs/03-ai/mastery-and-misconceptions.md`

## 背景

`mastery_states` 表的字段已定（ADR-0003），但 `mastery_score` 如何计算、ASSESS 的 REMEDIATE/ADVANCE guard 阈值（ADR-0004 留待本 ADR）、误区标签如何产生和消费，都是空白。约束：确定性打分（LLM 不得直接决定掌握度）、冷启动是常态、试点数据量小、每个学习事件后在线增量更新、结果必须可向教师和家长解释。

## 候选方案

- **BKT 族**：可解释、在线更新便宜，但多信号证据（提示、实验、动画步骤）难以透明地进入隐状态语义，小数据下逐知识点拟合不稳。留作后期离线影子基准；
- **PFA 风格确定性复合公式（本决定）**：从计数 + 时间衰减直接计算，冷启动用 Beta 平滑，天然可解释、增量成本最低、易与先修图结合；
- **DKT/SAKT/AKT 深度族**：小数据下脆弱、不可解释、与"事件级在线更新"冲突，复现研究表明其优势不稳健。不用于生产；
- **Elo/Glicko**：语义是"评分变动"而非概念掌握，不适合作为主引擎，后期可作题目难度校准的辅助层。

## 决定

1. **掌握度由 PFA 风格确定性公式计算**：Beta 平滑成功率 × 提示惩罚 × 活跃误区惩罚，与经时间衰减的前值加权混合，并被先修掌握度封顶（防止地基不稳时下游虚高）。公式与默认参数见规格文档；
2. **ASSESS 出口决策使用滞后区间**：进入掌握 `θ_enter = 0.85`，跌出掌握 `θ_exit = 0.75`，先修 gate `0.80`，且要求近期测评窗口内 ≥3 题、正确率 ≥80%。消除阈值边界震荡；
3. **复习调度与掌握度分离**：`next_review_at` 由独立的确定性分层调度器写入（v1 按掌握度分档定间隔，后续可平滑替换为 FSRS），调度器只决定"何时复习"，永不决定"是否掌握"；
4. **下一知识点推荐基于 ready-to-learn 图选择**：只推荐先修全部过 gate 的节点（ZPD 的计算化），配合双队列（复习队列/新学队列）的确定性优先级规则；
5. **LLM 只做误区标注，永不打分**：封闭核心标签集（三组：AI 系统本质 / ML 流程误解 / 学习与评价误解）+ 必须引用学生原话作为证据 + 允许弃权 + `CANDIDATE_NEW_LABEL` 提案通道。提案标签不直接驱动状态机，经人工评审后才能进入本体；
6. **误区标签有生命周期**：`misconception_tags` 从字符串数组演进为对象数组（tag、status、first_seen_at、last_seen_at），只有 `active` 状态的标签进入掌握度惩罚和 guard；REMEDIATE 完成且针对性测评通过后标记 `resolved`。JSONB 形状变化，无需 SQL 迁移。

## 原因

- 全部教学决策保持确定性、可审计，符合"模型只负责表达和受控工具调用"的架构原则；
- PFA 风格公式对冷启动（Beta 平滑）、小数据（无需拟合）、在线更新（计数增量）、可解释性（每个因子有名字）四项约束同时给出最好答案；
- 滞后区间和 ready-to-learn gate 有掌握学习（80–90% 高标准）和 K12 知识图谱平台的实证支持；
- 误区检测的文献共识是"约束任务"：封闭标签 + 证据引用 + 弃权，比开放生成可靠得多，且保证跨学生统计口径稳定。

## 后果

- 掌握度更新为纯逻辑，按测试规则必须在实现 PR 内附带单元测试；
- 所有数值参数（θ、衰减率、惩罚系数、调度间隔）为初始默认值，需建立评测集后校准；参数放配置不硬编码；
- 严重误区清单（直接触发 REMEDIATE 的标签）与课程绑定，放课程配置而非代码常量；
- `gradeAnswer`/`updateMisconception` 工具的输出契约由规格文档约束；
- BKT+forgetting 影子基准和 FSRS 替换是后期工作，各需独立评估。

## 验证方式

- 单元测试：公式的冷启动值、单调性（连续答对上升/连续求助下降）、先修封顶、滞后区间不震荡；
- 回放测试：从 `learning_events` 重算 `mastery_states` 与线上值一致（事件流是事实源，掌握度是导出值）；
- 试点后校准：用真实事件流离线扫描参数，检查 REMEDIATE/ADVANCE 决策与教师判断的一致率。
